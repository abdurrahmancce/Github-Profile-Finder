# 🚀 Github-Profile-Finder

A sleek and modern GitHub Profile Finder built with **HTML, CSS, and JavaScript**.  
Search any GitHub username and instantly explore profile details, repositories, followers, and more using the GitHub REST API.

Designed with a futuristic **glassmorphism + terminal-inspired UI** for a clean and immersive experience.

# ✨ Features

- 🔍 Search any GitHub profile instantly
- 👤 View profile information
- 📦 Display public repositories
- ⭐ Show repository stars and forks
- 👥 Followers & following count
- 📝 User bio, website, location & social links
- 💾 Last searched username saved using `localStorage`
- ⚡ Fast and responsive UI
- 🎨 Modern glassmorphism design
- 📱 Fully responsive for all devices
- ⌨️ Enter key support for quick search
- ❌ Beautiful error handling UI
- 🌙 Dark futuristic theme

# ⚙️ How It Works

1. User enters a GitHub username
2. App fetches data from GitHub API
3. Profile information is displayed dynamically
4. Repositories are sorted and rendered
5. Last searched username is saved locally
